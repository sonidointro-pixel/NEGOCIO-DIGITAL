# Roadmap inicial — NEGOCIO DIGITAL

## Fase 0 — Preparación
- Crear estructura de carpetas y documentación. (Hecho)
- Decidir stack.

## Fase 1 — MVP (0-4 semanas)
- Scaffold backend mínimo: API de ejemplo con un endpoint `GET /status`.
- Scaffold frontend mínimo: página index con conexión a API.
- Autenticación básica (email/password).

## Fase 2 — Funcionalidades centrales (4-12 semanas)
- CRUD de productos/servicios.
- Integración de pago.
- Panel administrativo.

## Fase 3 — Producción
- CI/CD, despliegue, monitorización.
- Escalado y optimizaciones.

Notas: adaptar tiempos según prioridad y recursos disponibles.
